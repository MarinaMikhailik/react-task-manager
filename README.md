# 📝 React Todo App

Простое и аккуратное приложение списка задач, созданное на React.
Проект сделан как pet-проект для практики и демонстрации на технических собеседованиях.

## 🚀 Демо

> (сюда можно добавить ссылку на Vercel / Netlify позже)

---

## ✨ Возможности

* ➕ Добавление задач
* ❌ Удаление задач
* ⌨️ Добавление по Enter
* 🚫 Блокировка кнопки при пустом вводе
* 📊 Счётчик задач
* 🧠 Обработка состояния через React Hooks
* 🎨 Современный UI (карточка, тени, hover-эффекты)

---

## 🛠️ Технологии

* React (Hooks)
* JavaScript (ES6+)
* CSS (без библиотек)
* Create React App

---

## 📁 Структура проекта

```
src/
│
├── components/
│   ├── Header.jsx        # Инпут + кнопка добавления
│   ├── TaskList.jsx     # Список задач
│   └── TaskItem.jsx     # Отдельная задача
│
├── App.js               # Основная логика
├── App.css              # Стили
└── index.js
```

## ⚙️ Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/react-todo-app.git

# Перейти в папку
cd react-todo-app

# Установить зависимости
npm install

# Запустить проект
npm start
```

Приложение откроется на:

```
http://localhost:3000
```

---

## 🧩 Как это работает

* Состояние задач хранится в `useState`
* Добавление задачи:

  * проверяется `trim()`
  * добавляется в массив
* Удаление задачи:

  * через `filter`
* Компоненты разделены по ответственности:

  * `Header` — ввод
  * `TaskList` — список
  * `TaskItem` — элемент
